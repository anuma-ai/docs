# Anuma

Build AI-powered apps with a unified API, persistent memory, and user-owned encryption.

Anuma gives you access to models from OpenAI, Google, Anthropic, xAI, and
open-source providers through a single API. Switch models without changing your
code, store semantic memories that persist across sessions, and keep user data
encrypted with keys only they control.

## SDK

For React and React Native apps. The SDK provides hooks that handle streaming,
authentication, and state management.

```bash
npm install @reverbia/sdk
```

```tsx
import { useChat } from "@reverbia/sdk/react";

const { sendMessage, isLoading } = useChat({
  getToken: async () => authToken,
  onData: (chunk) => setResponse((prev) => prev + chunk),
});

await sendMessage({
  messages: [{ role: "user", content: "Hello!" }],
  model: "gpt-4o-mini",
});
```
