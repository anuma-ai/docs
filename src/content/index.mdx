# Anuma

Build AI-powered apps with a unified API, persistent memory, and user-owned encryption.

Anuma is an AI platform that gives you access to models from OpenAI, Google,
Anthropic, xAI, and open-source providers through a single API. Built-in support
for streaming, persistent memory, tool calling, and client-side encryption.

```bash
npm install @reverbia/sdk
```

## Why Anuma?

- **One API, many models** — Switch between GPT-4, Gemini, Claude, DeepSeek, and
  more without changing your code
- **Private Memory Layer** — Semantic memory that persists across sessions,
  encrypted with user-controlled keys
- **Streaming built-in** — Real-time responses with hooks that handle state
  management
- **Client & server tools** — Define functions the model can call, with support
  for both client-side and server-side execution
- **React & React Native** — First-class support for web and mobile

## Quick Example

```tsx
import { useState } from "react";
import { useChat } from "@reverbia/sdk/react";

function Chat() {
  const [response, setResponse] = useState("");
  const { isLoading, sendMessage } = useChat({
    getToken: async () => getAuthToken(),
    onData: (chunk) => setResponse((prev) => prev + chunk),
  });

  return (
    <div>
      <p>{response}</p>
      <button
        onClick={() =>
          sendMessage({
            messages: [{ role: "user", content: "Hello!" }],
            model: "gpt-4o-mini",
          })
        }
      >
        {isLoading ? "Thinking..." : "Send"}
      </button>
    </div>
  );
}
```

The SDK handles streaming, authentication, and state management so you can focus
on building your product.
