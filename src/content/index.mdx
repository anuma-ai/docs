# Anuma

Build AI-powered apps with a unified API, persistent memory, and user-owned encryption.

Anuma gives you access to models from OpenAI, Google, Anthropic, xAI, and
open-source providers through a single API. No juggling provider keys, no
managing multiple SDKs. Switch models without changing your code, store semantic
memories that persist across sessions, and keep user data encrypted with keys
only they control.

Your users can pay for their own inference directly — you don't have to subsidize
API costs or manage billing.

## SDK

For React and React Native apps. The SDK provides hooks for chat, memory, tool
calling, image generation, web search, and encryption.

```bash
npm install @reverbia/sdk
```

```tsx
import { useChat } from "@reverbia/sdk/react";

const { sendMessage, isLoading } = useChat({
  getToken: async () => authToken,
  onData: (chunk) => setResponse((prev) => prev + chunk),
});

await sendMessage({
  messages: [{ role: "user", content: "Hello!" }],
  model: "gpt-4o-mini",
});
```

## Portal API

For other languages or direct HTTP access. OpenAI-compatible, so you can use
existing tools and libraries. Chat completions with streaming, embeddings, tool
calling, and server-side tools for image generation and web search.
