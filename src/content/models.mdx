# Models

The Portal API supports multiple AI models from different providers. When
sending a message, you specify which model to use. Each model has different
capabilities, pricing, and performance characteristics.

Models fall into several categories based on their modalities:

- **LLM** — Text generation and conversation
- **Vision** — Image understanding and analysis
- **Reasoning** — Extended thinking for complex problems
- **Image** — Image generation
- **Code** — Programming and code generation
- **Audio** — Speech and audio processing

Some models support multiple modalities. For example, vision-capable LLMs can
both generate text and analyze images.

See the [full list of available models](/reference/models-list).
