# Anuma Docs

Documentation for [Anuma](https://anuma.ai) — a unified AI SDK for building
AI-powered apps. Anuma provides a single API to access models from OpenAI,
Google, Anthropic, xAI, and open-source providers, with built-in conversation
memory, user-owned encryption, and local-first data storage.

https://docs.zetachain.workers.dev/

## Development

```
pnpm install
pnpm dev
```

## Deployment

```
pnpm deploy:cloudflare
```
